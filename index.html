1.DNS 如何选择：

在国内一般使用我提供的几个国内 DNS 服务器都可以，如果身处海外的话建议选择海外的公共 DNS 服务器。另外经过测试，114DNS、百度 DNS、腾讯 DNS 在海外大部分地区都可以使用，而 阿里DNS 在海外很多地区连通性不太好。

2.什么是 DoT 和 DoH：

DoT 和 DoH 都是加密DNS的一种方式，区别在于它们采用不同的协议和端口，两个都是域名解析安全扩展协议的一种。
概念：
DoT 全称 DNS over TLS，它使用 TLS 来传输 DNS 协议。
DoH 全称 DNS over HTTPS，它使用 HTTPS 来传输 DNS 协议。

两个协议原理是相同的，都是通过加密传输用户和 DNS 服务器之间的 DNS 消息，起到防止中间用户窃听和域名查询隐私泄漏的作用。相对来说 DoH 更通用一些。

3.设置 DNS 和 DoT/DoH 的方法：

Windows 设置 DNS 的方法：

打开控制面板 > 网络和共享中心 > 更改适配器设置。
右键单击您的网络连接，然后选择属性。
选择“Internet 协议版本 4 (TCP/IPv4)”并点击属性。
输入所需的DNS服务器地址。

MacOS 设置 DNS 的方法：

转到系统偏好设置 > 网络。
选择您的网络连接并点击高级。
转到DNS标签页并点击+按钮添加新的DNS服务器。

路由器 设置 DNS 的方法：

访问您的路由器管理界面（通常是在您的网页浏览器中输入路由器的IP地址）。
查找DNS设置（这因路由器型号而异）。
输入所需的DNS服务器地址。

Chrome 浏览器设置 DoH 的方法：

打开 Chrome 浏览器，点右边的三个点 打开菜单，选中 设置 选项。
选中左边的 隐私设置和安全性，然后点击右边找到的 安全。
在打开的页面中，选中 使用安全 DNS 后面的滑块，再点击下面 使用，下拉框里选择自定义，然后在下面 文本框 中输入自己找到的 DoH 服务器。

Edge 浏览器设置 DoH 的方法：

打开 Edge 浏览器，点右上角三个点 ... 打开菜单，点击 设置 选项。
在打开的 设置 页面里，选择左边的 隐私、搜索和服务 选项，然后下拉找到 安全性，选中 使用安全的 DNS 指定如何查找网站的网络地址 后面的滑块，再点击 请选择服务提供商 前面的单选框，在下面的文本框里可以输入自己找到的 DoH 服务器地址。
